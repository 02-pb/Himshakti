# PahadiKart

A mobile-first direct-to-consumer (D2C) landing page for HimShakti Food Processing Unit, showcasing Himalayan food products with WhatsApp-based ordering and shopping cart functionality.

---

## Tech Stack

* React.js (Frontend UI Library)
* React Router DOM (Routing & Navigation)
* HTML5 & CSS3 (Structure & Styling)
* JavaScript (ES6+) (Core Logic)
* Bootstrap / Tailwind CSS (Responsive UI Design)
* WhatsApp API (Order Checkout Integration)
* Git & GitHub (Version Control & Deployment)
